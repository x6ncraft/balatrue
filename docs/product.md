# 产品与规则

## 一句话定义

每天用最多六次出牌机会，猜出藏在牌背后的《小丑牌》小丑卡。

## 名称

- 英文名：`Balatrue`
- 中文名：`猜丑牌`
- 英文副标题：`An unofficial daily Balatro Joker puzzle`
- 中文副标题：`非官方《小丑牌》每日竞猜`

`Balatrue` 把 `Balatro` 与 `true` 合在一起：既能直接唤起《小丑牌》，也对应从真假线索
中找出正确答案。中文名用三个字直接说明动作，并借用「小丑牌」里的「丑牌」形成轻松的
文字游戏。页面副文案为「猜猜今天是哪张小丑牌」。

## 首版模式

### 今日谜牌

- 每个北京时间自然日共享一个答案。
- 最多六次出牌机会。
- 150 张答案按固定种子洗牌，一个周期内不重复；新数据版本不会改变当天答案。
- 本地保存当日进度、胜负、连胜、胜率与平均出牌数。

### 无尽牌局

- 使用本地洗牌袋，整轮 150 张目标不重复。
- 每局结束后可立即再开一局，不影响每日成绩。

## 五项线索

| 线索           | 玩家看到的内容                                             | 反馈规则                                 |
| -------------- | ---------------------------------------------------------- | ---------------------------------------- |
| 稀有度         | 普通、罕见、稀有、传奇                                     | 相同为绿；否则箭头指向答案               |
| 基础价格／获取 | 商店价格；传奇显示灵魂牌                                   | 同价为绿；数字显示方向；特殊获取单独比较 |
| 主效果         | 筹码、+倍率、×倍率、经济、重触发、机制                     | 集合相同为绿；至少一个标签相同为黄       |
| 生效时点       | 计分、出牌、留手、弃牌、选盲注、商店、回合结束、持续       | 集合相同为绿；至少一个时点相同为黄       |
| 依赖条件       | 花色、点数、人头牌、牌型、小丑、消耗牌、牌组、金钱、回合等 | 具体值相同为绿；同族不同值为黄           |

稀有度按官方简中顺序：`普通 → 罕见 → 稀有 → 传奇`。

首版数据校验要求 150 张牌的完整五维签名互不重复，避免出现仅靠公开线索无法区分的二选一。

## 线索字典与小丑图鉴

- 线索字典列出五项线索的全部顶层枚举，以及当前 150 张牌实际使用的依赖细项。它属于规则说明，随时查看都不影响成绩。
- 小丑图鉴包含 150 张牌的官方中英文名、卡图、稀有度、基础价格，以及 Balatrue 整理的三项竞猜分类；支持中英文、拼音搜索与四类筛选。
- 每日谜牌进行中打开图鉴前必须确认；确认后牌局仍可继续、保存和分享，但不写入已玩、胜率、连胜与平均出牌数。
- 已结束的每日局查看图鉴不会追溯取消成绩；无尽牌局本来就不计战绩，因此直接打开。
- 图鉴辅助局的分享首行带 `🔎` 标记，避免和独立完成的计分局混淆。

## 输入联想

- 同时索引官方简中名、英文名、中文全拼和拼音首字母。
- 中文界面以中文名为主、英文名为辅；英文界面反向展示。
- 候选项展示缩略图，不提前泄露稀有度和效果。
- 支持方向键、Enter、Escape 和中文输入法 composition，已经猜过的牌不可重复提交。

## 反馈和可达性

- `✓` 完全吻合：绿色。
- `◐` 部分吻合：黄色。
- `×` 不吻合：暗红或中性灰。
- 数值不相同时，`↑ / ↓` 始终指向正确答案。
- 颜色之外始终保留图标、文本和边框差异。
