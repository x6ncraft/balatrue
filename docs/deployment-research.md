# 域名、部署与基础统计

核验日期：2026-07-19。本文件记录会变化的域名状态、价格与平台政策；本轮只完成调研，没有购买
域名、创建云资源或部署站点。

## 建议结论

Balatrue 适合放在 x6n 的统一品牌域名下：

- 品牌根域：`x6n.me`
- 产品正式地址：`balatrue.x6n.me`
- 托管：Tencent EdgeOne Pages（Makers 免费档）
- 首发服务区：全球可用区，不含中国大陆
- 首发统计：EdgeOne 自带的请求量、流量、地区和状态码；产品事件以后再接

`x6n.me` 是当前最均衡的选择：名称长度已经达到 `.me` 注册局允许的三字符下限；注册局实时查询
显示它尚未注册且不是溢价词；Porkbun 对该精确名称的本次报价为注册、续费和转入均 `$17.27 / 年`。
`.me` 也在工信部当前公布的可备案顶级域范围内。一个根域可以免费创建任意合理数量的子域，今后
可继续使用 `clerk.x6n.me`、`notes.x6n.me` 等地址，不必为每个小产品再买域名。

域名可用性不等于预留，结账前仍需重新确认。若明确准备办理 ICP，优先在支持 `.me` 且具备境内
许可的注册服务商检查 `x6n.me` 的精确首年、续费和转入价格；先在海外注册商购买，日后可能还要
经历 60 天转移锁、实名与转入流程。

所以 `x6n.x` 不能注册。这里的 `x` 只能作为“待选后缀”的占位符。

### `.io` 和 `.ai` 怎么选

`x6n.io` 适合把 x6n 明确塑造成开发者产品品牌。它比 `.me` 每年多约 `$34.53` 续费，而且 `.io`
是与英属印度洋领地代码绑定的国别域；IANA 当前仍正常委派，但国别代码发生变化时存在注册局政策
调整的长期可能。它可以作为偏技术的防御域或跳转域，不建议优先于 `x6n.me`。

`x6n.ai` 更像一家 AI 公司。它的普通年费约为 `.me` 的 4.8 倍，很多注册商按两年起售；如果 x6n
以后确实集中做 AI 产品，这笔费用有品牌价值。Balatrue 本身不是 AI 产品，现在买它主要是在为未来
品牌占位，和本项目上线没有技术上的必要关系。

按 2026-07-19 的工信部公示，两者都不在当前可备案后缀范围内。即使部署在境外节点能够从中国
访问，也不能作为现行大陆节点备案主域。若只维护一个根域，仍选 `x6n.me`；若愿意为未来技术品牌
多保留一个，优先级为 `x6n.io` 高于 `x6n.ai`。

### 价格与备案来源

- `.me` 注册局的[实时名称查询](https://aisearch.domain.me/main/1/)显示 `x6n.me` 可注册、非溢价；
  [注册局费用表](https://domain.me/files/online%20service%20and%20fee%20schedule.pdf)给出的批发基准
  为 `€10 / 年`，实际零售价由注册商决定。
- `.im` 注册局的[官方价格表](https://www.nic.im/public/pricing.mth)列出标准名称 `£40 / 年`。
- 其余美元价格来自 [Porkbun 域名价格页](https://porkbun.com/products/domains/)和对应名称的实时
  查询；例如 [`x6n.me`](https://porkbun.com/checkout/search?q=x6n.me)、
  [`x6n.io`](https://porkbun.com/checkout/search?q=x6n.io) 与
  [`x6n.ai`](https://porkbun.com/checkout/search?q=x6n.ai)。精确名称是否溢价以结账页为准。
- `x6n.hk` 只由 [HKIRC WHOIS](https://www.hkirc.hk/en/our-support-support-services-whois-search/)
  确认未检出注册；这不等于已经核验零售资格、材料和结算价。
- 可备案后缀与获批注册服务机构应在购买和备案时分别按[工信部域名管理机构公示](https://domain.miit.gov.cn/managerdns/index)
  复核；“后缀支持”不代表任何海外注册商都能直接办理备案。

注册域名也不会取得 `Balatrue` 名称、Balatro 文本或卡图的权利。域名选择和素材权利仍是两个
独立问题。

## 托管方案

### 首选：EdgeOne Pages

[EdgeOne Pages 价格页](https://pages.edgeone.ai/pricing)当前提供长期 `$0 / 月` 免费档，支持 Git
自动部署、全球 CDN、自定义域名和免费 HTTPS。免费额度包括每月 500 次构建、5 GB 项目存储和
200 个自定义域名；官方没有公布静态请求或 CDN 流量硬上限，同时保留以后调整额度的权利。
[额度说明](https://pages.edgeone.ai/document/limits-and-quotas)

Balatrue 的部署契约很简单：

```text
Region:            Global, excluding Chinese mainland
Install:           bun install --frozen-lockfile
Build:             bun run build
Output:            dist
Production branch: main
Custom domain:     balatrue.x6n.me
Node:              22.17.1
```

平台可以直接[导入 GitHub 仓库](https://pages.edgeone.ai/document/importing-a-git-repository)，推送
`main` 后自动构建。首发选择“不含中国大陆”，无需 ICP，也减少上线前置；境外邻近节点通常可以
从大陆访问，但跨境网络随地区、运营商和时段波动，不能承诺稳定顺滑。

该服务区的 EdgeOne 默认项目域名从中国大陆访问会返回 401；大陆实网验收必须使用已经绑定的
`balatrue.x6n.me` 自定义域名，不能拿默认预览地址判断站点可用性。未来启用大陆节点时，还要按
平台当时能力确认原项目能否切换服务区，或是否需要建立新项目迁移。

平台只保留最近三个成功部署产物；Git 历史完整的静态站可以重新构建旧提交，因此影响可控。

## 中国大陆访问边界

[EdgeOne 域名与区域说明](https://pages.edgeone.ai/document/domain-overview)明确：全球不含中国大陆的
服务区无需 ICP；使用中国大陆或全球含大陆节点必须备案。没有备案时，境外托管只能以真实三网
测试判断可用程度，任何平台都无法保证大陆长期顺滑。

ICP备案解决网站接入资格，也不自动解决网页游戏的出版分类。现行《网络出版服务管理规定》把
通过网络向公众提供的游戏列入网络出版物；Balatrue 免费、无账号、纯前端、粉丝自制，有利于
说明实际风险较低，但法规文本没有给个人非盈利网页小游戏一个明确豁免。启用大陆节点前，应把
实际页面、玩法、素材边界和非盈利事实交给接入商合规团队，取得书面分类意见。

- [网络出版服务管理规定](https://www.nppa.gov.cn/xxfb/zcfg/bmgz/201602/t20160206_4403.html)
- [网络出版服务审批事项](https://www.nppa.gov.cn/bsfw/xksx/cbfxl/wlcbfwspsx/202210/t20221013_600725.html)

建议先发布境外节点版本，观察中国移动、联通、电信的实际访问；取得备案和内容分类结论后，再
决定切换 EdgeOne 全球含大陆服务区，或增加腾讯 COS + CDN。

## 基础统计

[EdgeOne 项目指标](https://pages.edgeone.ai/document/metric-analysis)无需页面 SDK，可以查看请求量、
流量、国家或地区和状态码，足够回答“有没有访问、主要来自哪里、是否有错误”。一次页面打开会
请求多份 HTML、JavaScript、字体和图片，所以请求数不能写成 PV 或访客数。

若需要真实 PV/UV，可后续接免费的 Cloudflare Web Analytics；若需要开局、猜测、过关、分享和
图鉴打开等产品事件，再考虑 PostHog。PostHog 当前每月前 100 万个产品事件免费。
[PostHog 价格](https://posthog.com/)

最小事件建议：

- `game_started`
- `guess_submitted`
- `game_finished`
- `collection_opened`
- `outbound_link_clicked`

只上报匿名事件名、语言、模式、猜测次数与是否过关，不上传答案、具体猜测、自由文本或本地存档。
接入前更新页面隐私说明；统计脚本不可用时，游戏仍应完整可玩。

## 推荐执行顺序

1. 用户在候选中确认品牌根域；结账前复核可用性、首年、续费、溢价和备案计划。
2. 注册一个根域并配置稳定的联系邮箱；`balatrue` 子域不产生额外域名费用。
3. EdgeOne 导入公开仓库，以“全球不含中国大陆”发布 `balatrue.x6n.me`。
4. 用中国移动、联通、电信及香港、日本、北美网络验收 DNS、TLS、首屏、卡图和缓存加载。
5. 连续观察 3 至 7 天的真实质量和基础指标，再判断是否需要事件 SDK。
6. 大陆节点等待 ICP 可行性和网页游戏内容分类的书面结论。

购买域名、创建托管项目和切换生产域名都属于后续外部动作，执行前由用户确认。
