# 路线图

当前可玩版本已经进入发布准备阶段。执行状态和外部动作以 `release-checklist.md` 为准，本页只保留
方向，不重复完整清单。

## 接下来

- 用户确认后，从当前文件树建立干净根提交或新公开仓库，避免把历史中的完整英文效果与解锁原文
  一并公开。
- 从干净 checkout 运行完整回归，确认 MIT 许可证、素材说明、逐卡来源记录和第三方许可证均进入
  发布产物，本地受限审计字段没有进入 Git 或 `dist/`。
- 由用户选择域名与托管方案，再分别处理境外站点和中国大陆节点；源码公开不自动触发部署。
- 继续保留清楚的非官方、免费非商业、素材排除 MIT、残余风险和下架说明；若收到合理权利要求，
  及时调整、替换或移除相关内容。

创建仓库、改写历史、切换可见性、购买域名和创建生产站点均等待用户明确确认。

## 上线后优先观察

- 新玩家能否在不读帮助的情况下理解五项线索。
- 六次机会的完成率、平均猜测次数和图鉴辅助率。
- 375px 手机上的连续多次猜测是否仍便于逐行比较。
- 中英文牌名、触发时机和依赖条件是否还有容易误解的个案。

## 可能的后续版本

- 在明确告知并取得适用同意后，接入少量匿名游戏事件，观察开局、完成和分享趋势。
- 根据真实反馈修订 `g3` 玩家线索；任何语义改动都伴随版本升级、存档迁移与 150 张唯一性检查。
- 增加可分享的自定义练习种子，但继续保持无账号、无排行榜和纯前端边界。
- 在权利边界允许的范围内补充更清楚的资料说明；完整上游原文继续只留在本地审计文件中。

没有真实需求和验证依据的功能不提前排期。
